# phobos_web_ctrl

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Runs ZMQ to WebSocket shim
Must install nodemon firsts:
```
npm i -g nodemon
```

```
nodemon src/zmq_shim.js
```
